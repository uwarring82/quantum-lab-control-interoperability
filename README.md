# Kompass-Dossier: Open-Source Quantum Laboratory Control

**Control, Data, and Interoperability in the ARTIQ Ecosystem**

This repository hosts a community landscape dossier mapping the open-source ecosystem for quantum laboratory control built around ARTIQ/Sinara. It covers control-plane interoperability, data management and provenance, and adaptive optimisation workflows.

## Purpose

The dossier identifies structural overlaps between independent projects (ARTIQ, QUARTIQ, Atomiq ONE, DAX, OQD, Oxford Ion Trap Group tools), analyses their architectural differences, and poses concrete interoperability questions to the community. It does not prescribe a preferred stack or define standards — it maps the terrain and invites coordination.

## How to contribute

See [CONTRIBUTING.md](CONTRIBUTING.md) for guidelines. In brief:

- **Factual corrections** (errors, outdated status, missing projects): open an issue or pull request.
- **Alternative perspectives** (disagreements on framing, emphasis, or analysis): open a discussion or issue — documented diversity of views is preferred over forced consensus.
- **New interoperability proposals**: welcome as issues for discussion before integration.

## Stewardship

This repository is stewarded as a neutral coordination space for a landscape dossier on interoperability in the ARTIQ ecosystem. It is **not** the governance home of ARTIQ, Sinara, DAX, Atomiq ONE, OQD, Oxford tools, QUARTIQ, or any other project described herein. Contributions are welcome in the form of factual corrections, references, clarifications, and documented alternative perspectives.
